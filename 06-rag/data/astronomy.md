## 太阳系的中心

# 太阳是太阳系的中心天体，占太阳系总质量的 99.86%。它是一颗黄矮星，主要由氢和氦组成。

## 水星的特征

# 水星是距离太阳最近的行星，也是太阳系中最小的行星。由于没有大气层，它的昼夜温差极大。

## 金星的大气

# 金星的大气层非常浓厚，主要成分是二氧化碳，导致了严重的温室效应，使其表面温度甚至高于水星。

## 地球的独特性

# 地球是目前已知唯一孕育生命的天体。它拥有液态水、含氧大气层和强磁场。

## 火星的红色外观

# 火星被称为“红色星球”，其表面的红色源于大量的氧化铁（铁锈）。它拥有太阳系最高的山脉——奥林帕斯山。

## 木星的质量

# 木星是太阳系中最大的行星，其质量是其他所有行星总和的 2.5 倍。它拥有著名的大红斑气旋。

## 土星的光环

# 土星以其壮观的光环系统闻名，光环主要由冰颗粒、岩石残骸和尘埃组成。

## 天王星的自转

# 天王星的自转轴几乎与轨道平面平行，就像是在轨道上“滚动”一样。它是一颗冰巨星。

## 海王星的发现

# 海王星是通过数学计算而非直接观察首先被预言存在的行星。它的大气中存在强烈的风暴。

## 冥王星的分类

# 冥王星曾被视为第九大行星，但在 2006 年被国际天文学联合会重新定义为“矮行星”。

## 银河系的形状

# 银河系是一个棒旋星系，拥有数千亿颗恒星。太阳位于银河系的猎户座旋臂上。

## 黑洞的概念

# 黑洞是时空展现出极端强大引力的区域，其引力大到连光都无法逃逸的。

## 哈勃空间望远镜

# 哈勃望远镜自 1990 年发射以来，为我们提供了大量关于宇宙膨胀和遥远星系的珍贵图像。

## 詹姆斯·韦伯空间望远镜

# 作为哈勃的继任者，韦伯望远镜主要在红外波段工作，旨在探测宇宙形成初期的第一批星系。

## 月球的同步自转

# 由于潮汐锁定，月球始终以同一面朝向地球。这就是为什么我们从地球上看不到月球的背面。

## 超新星爆发

# 超新星是某些恒星演化接近末期时经历的一种剧烈爆炸，其亮度可以短时间内超过整个星系。

## 脉冲星的本质

# 脉冲星是快速自转的中子星，能够定期发射电磁脉冲信号。

## 宇宙大爆炸理论

# 大爆炸理论是描述宇宙起源的主流科学模型，认为宇宙起源于约 138 亿年前的一个极高密度和温度的点。

## 极光的成因

# 极光是由来自太阳的带电粒子流（太阳风）进入地球磁场，并与高层大气中的原子碰撞产生的。

## 彗星的组成

# 彗星通常被称为“脏雪球”，主要由冰、尘埃和岩石组成。当靠近太阳时，会形成长长的彗尾。

## 小行星带

# 小行星带位于火星和木星轨道之间，聚集了数以百万计的小型岩石天体。

## 柯伊伯带

# 柯伊伯带是海王星轨道之外的一个盘状区域，充满了冰冻天体，是许多短周期彗星的来源。

## 奥尔特云

# 奥尔特云是一个包围整个太阳系的球壳状区域，被认为是长周期彗星的诞生地。

## 引力波的探测

# 引力波是时空曲率的涟漪，由爱因斯坦预言。2015 年，LIGO 首次探测到了双黑洞合并产生的引力波。

## 光年的定义

# 光年是光在真空中沿直线传播一年所经过的距离，约为 9.46 万亿公里。

## 恒星的颜色与温度

# 恒星的颜色反映了其表面温度：蓝色恒星温度最高，红色恒星温度相对较低。

## 赫罗图

# 赫罗图是反映恒星光谱类型与光度之间关系的图表，是研究恒星演化的重要工具。

## 主序星阶段

# 恒星生命中大部分时间处于主序星阶段，此时其核心正进行着氢核聚变。

## 红巨星的形成

# 当恒星核心的氢耗尽后，它会膨胀并冷却，演化为红巨星。

## 白矮星的结局

# 低质量恒星（如太阳）在演化末期会抛射外层，核心坍缩为高密度的白矮星。

## 中子星的密度

# 中子星是巨型恒星发生超新星爆发后的残核，其密度极高，一茶勺的物质就重达数亿吨。

## 变星的特性

# 变星是指亮度随时间发生变化的恒星，如造父变星，常被用作宇宙中的“标准烛光”。

## 系外行星

# 系外行星是指位于太阳系之外、环绕其他恒星运行的行星。目前已发现数千颗。

## 宜居带

# 宜居带是指恒星周围的一定距离范围，在该范围内行星表面可能存在液态水。

## 类星体

# 类星体是宇宙中最明亮、最遥远的天体之一，其能量来源于中心超大质量黑洞吞噬物质。

## 暗物质

# 暗物质是一种不发射、不吸收电磁辐射的物质，通过其引力效应被推断存在，占宇宙总质能的约 27%。

## 暗能量

# 暗能量是一种充斥在宇宙空间的未知能量形式，被认为导致了宇宙的加速膨胀。

## 宇宙微波背景辐射

# 这是大爆炸留下的“余温”，是充满整个宇宙的微弱微波信号，为大爆炸理论提供了有力证据。

## 埃德温·哈勃的发现

# 哈勃发现远处的星系都在远离我们，且距离越远退行速度越快，这表明宇宙正在膨胀。

## 凌日法探测行星

# 凌日法是通过监测恒星亮度的微小下降来探测环绕其运行的行星的一种方法。

## 视向速度法

# 又称多普勒频移法，通过观察恒星光谱的周期性移动来探测行星的引力扰动。

## 伽马射线暴

# 伽马射线暴是宇宙中最剧烈的爆炸，产生的伽马射线光度极高。

## 猎户座大星云

# 猎户座大星云是一个著名的恒星诞生区，距离地球约 1300 光年。

## 仙女座星系

# 仙女座星系是距离银河系最近的大型星系，预计在几十亿年后将与银河系合并。

## 局部星系群

# 银河系、仙女座星系以及其他约 50 个较小的星系共同组成了局部星系群。

## 拉尼亚凯亚超星系团

# 这是我们所在的超星系团，包含约 10 万个星系，横跨 5 亿光年。

## 黄道十二宫

# 黄道是地球绕太阳公转轨道平面在天球上的投影。沿黄道分布的 12 个星座被称为黄道十二宫。

## 岁差运动

# 地球自转轴在空间中缓慢的圆锥形摆动被称为岁差，周期约为 26000 年。

## 朔望月与恒星月

# 朔望月是月相变化的一个周期（约 29.5 天），恒星月是月球绕地球转动一周的真实周期（约 27.3 天）。

## 宇宙的最终演化

关于宇宙的结局，科学家提出了“大冻结”、“大撕裂”或“大挤压”等多种假说。
